# Triage Labels

The skills speak in terms of five canonical triage roles. This file maps those roles to the actual label strings used in this repo's issue tracker.

| Skill role | GitHub label | Meaning |
| --- | --- | --- |
| `needs-triage` | `agent:needs-triage` | Maintainer needs to evaluate this issue |
| `needs-info` | `agent:needs-info` | Waiting on reporter for more information |
| `ready-for-agent` | `agent:ready-for-agent` | Fully specified, ready for an AFK agent |
| `ready-for-human` | `agent:ready-for-human` | Requires human implementation |
| `wontfix` | `agent:wontfix` | Will not be actioned |

When a skill mentions a role, use the corresponding GitHub label from this table.
